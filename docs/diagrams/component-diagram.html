Backend API  [Container: Python 3.11 / FastAPI / Uvicorn]
Users Router
[Component: FastAPI APIRouter — routers/users.py]
Expone endpoints de sesión:
POST /users/login → emite JWT.
GET /users/me → retorna perfil del usuario autenticado.
Admin Router
[Component: FastAPI APIRouter — routers/admin.py]
Endpoints exclusivos Admin:
GET/POST /admin/users/*
PATCH activación/desactivación de usuarios. Requiere rol admin.
Patients Router
[Component: FastAPI APIRouter — routers/patients.py]
CRUD completo de pacientes:
GET/POST/PUT/DELETE /patients/*
POST /patients/{id}/predict → lanza predicción ML. Soporta paginación y búsqueda.
Auth Component
[Component: python-jose / passlib / bcrypt — auth.py]
Crea y verifica tokens JWT (HS256). Hashea y verifica contraseñas con bcrypt. Configura OAuth2PasswordBearer.
RBAC Guards
[Component: FastAPI Depends — dependencies.py]
Valida Bearer token en cada request. Aplica guards de rol:
get_current_active_admin
get_current_active_medico
ML Inference Engine
[Component: scikit-learn / joblib / pandas]
Carga model_pipeline.joblib (lazy load). Recibe niveles D1–D6 del paciente, ejecuta pipeline K-Means + Gradient Boosting y retorna perfil 0/1/2 de barreras de discapacidad.
CRUD Layer
[Component: SQLAlchemy ORM — crud.py / models.py]
Operaciones de base de datos para User y Patient. Incluye filtrado por owner_id (RBAC a nivel de dato), soft-delete y actualización de predicciones.
Database Session
[Component: SQLAlchemy — database.py]
Gestiona el engine y las sesiones de SQLAlchemy. Provee get_db() como dependency inyectable. Crea tablas en startup via Base.metadata.create_all().
PostgreSQL 15
[Container: Render Managed DB]
Tablas: users, patients. 3 instancias: dev / qa / prod.
model_pipeline.joblib
[Artifact: scikit-learn Pipeline]
Archivo binario del modelo ML (K-Means + Gradient Boosting). Generado por model/train_model.py. Incluido en el repositorio bajo backend/model/.
