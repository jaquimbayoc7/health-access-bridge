Health Access Bridge
[Software System]
Plataforma web para gestión clínica de pacientes con discapacidad y análisis predictivo de perfiles de barreras de acceso a la salud en zonas rurales. Incluye autenticación RBAC, CRUD de pacientes, predicción ML y dashboards analíticos.
PostgreSQL Database
[Software System]
Base de datos relacional managed en Render. Almacena usuarios, pacientes, historias clínicas y resultados de predicciones.
HybridModelDisability
ML Model
[Software System]
Modelo de Machine Learning embebido (K-Means + Gradient Boosting). Predice perfiles de barreras de discapacidad basado en niveles de actividad D1-D6.
OpenAI GPT-4 API
[Software System]
Servicio de IA generativa para orquestación de recomendaciones clínicas personalizadas. (HU-07 - Futuro)
GitHub Actions
[Software System]
Pipeline CI/CD automatizado. Ejecuta pruebas (pytest), build (Vite), y despliega a Render en 3 ambientes (dev/qa/prod) con gate de aprobación manual en producción.
Render Cloud Platform
[Software System]
Plataforma de hosting cloud. Despliega backend (Python/FastAPI), frontend (Node.js/Vite) y PostgreSQL managed en 3 ambientes independientes con auto-deploy desde GitHub.
