# HAB — UI Design Files

Diseños visuales del frontend de **Health Access Bridge** generados con Windsurf Pencil.

## Mockups exportados

Todas las pantallas están disponibles como imágenes PNG de alta calidad en la carpeta [`images/`](./images/).

## Screens incluidos

| # | Pantalla | Descripción | Rol | Archivo |
|---|----------|-------------|-----|---------|
| 01 | Login / Landing | Página de inicio con hero section + formulario de login por rol | Público | `bi8Au.png` |
| 02 | Dashboard | Métricas de pacientes, tabla de recientes, historial de predicciones | Médico | `717A2.png` |
| 03 | Pacientes | Listado con búsqueda, exportación Excel/PDF, acciones por fila | Médico | `i0uhw.png` |
| 04 | Predicciones | Formulario de nueva predicción + historial de resultados | Médico | `YPvRD.png` |
| 05 | Admin Panel | Formulario de creación de usuario + tabla de gestión con toggle | Administrador | `UpZqF.png` |
| 06 | Análisis | 4 métricas estadísticas + gráfico donut de perfiles | Médico | `HIxNY.png` |
| 07 | Guía Predictiva | Guía de interpretación de perfiles de barreras con ilustraciones | Médico | `Nj6le.png` |
| 08 | Perfil Funcional ICF | Generación de códigos ICF y recomendaciones según Resolución 113 de 2020 | Médico | `C45o8.png` |

## Archivo fuente

| Archivo | Descripción |
|---------|-------------|
| `hab-designs.pen` | Archivo maestro editable con todos los screens del sistema (Windsurf Pencil) |
